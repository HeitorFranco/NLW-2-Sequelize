# Instalando

## Clonando Projeto

```
$ git clone https://github.com/HeitorFranco/NLW-2-Sequelize.git

$ cd NLW-2-Sequelize
```

## Instalando Dependências

```
$ yarn
```

_or_

```
$ npm install
```

## Executando Aplicação

```
$ yarn start
```

_or_

```
$ npm start
```
